## End-point: Create a Food
### Method: POST
>```
>http://localhost:8080/api/foods
>```

|Content-Type|Value|
|---|---|
|Content-Type|application/json|


### Headers

|Content-Type|Value|
|---|---|
|x-access-token|{{accessToken}}|
